# 微信公众号内容发布合规规则

> 本规则基于微信公众平台运营规范、2025年10月AI内容标注新规，以及实际运营经验总结。
> 最后更新：2026-06-02

---

## 一、境外AI工具/软件（✅ 允许提及，比小红书宽松）

### 1.1 核心原则：可以提，客观写即可

微信公众号**没有禁止**提及或介绍境外AI工具。你可以：

- ✅ 在标题和正文中提及Claude、ChatGPT、Copilot、Midjourney等品牌名
- ✅ 分享使用心得和体验
- ✅ 做工具对比和测评
- ✅ 介绍功能和使用方法

### 1.2 写法建议

虽然微信对境外AI工具比较宽松，但以下写法仍然**建议避免**：

| 写法 | 建议 | 原因 |
|------|------|------|
| 「Claude使用指南」 | ✅ 可以 | 客观介绍 |
| 「ChatGPT帮我提升了3倍效率」 | ✅ 可以 | 分享真实体验 |
| 「5款AI编程工具对比」 | ✅ 可以 | 中性对比 |
| 「不用Claude你就OUT了」 | ⚠️ 避免 | 虚假紧迫感，可能触发夸大宣传 |
| 「AI工具排行榜第一名」 | ⚠️ 避免 | 无法证实的排名，触发虚假宣传 |

### 1.3 关键区别：微信 vs 小红书

| 对比项 | 微信公众号 | 小红书 |
|--------|-----------|--------|
| 标题提境外AI工具 | ✅ 可以 | ⚠️ 可以但必须中性，不能带安利语气 |
| 分享使用心得 | ✅ 正常内容 | ⚠️ 需注意措辞，避免安利感 |
| 强烈推荐语气 | ⚠️ 可能触发虚假宣传 | ❌ 高危，极易违规 |
| 教授如何访问境外服务 | ❌ 不建议 | ❌ 违规 |

## 二、AI内容标注（⚠️ 强制要求，2025年10月新规）

### 2.1 标注规则

- 使用AI辅助撰写的文章，**必须**在正文开头或结尾注明
- **未标注=违规**，处罚：限流→下架→封号
- AI辅助翻译/润色/排版**也建议标注**（非强制但更安全）

### 2.2 标注示例

**简洁版**：
```
本文由AI辅助撰写，内容仅供参考
```

**完整版**：
```
📝 声明：本文由AI辅助生成初稿，经人工审核、修改和补充后发布。
```

**在文末标注**：
```
---
本文使用AI工具辅助撰写，核心观点和内容均经人工审核确认。
```

### 2.3 腾讯官方建议

- 建议**优先使用国产AI工具**（混元、文心一言、讯飞星火等通过安全检测的大模型）
- 这是**推荐不是禁止**，用国外AI工具写文章本身不违规
- 标注了就没事，AI辅助创作是被允许的

## 三、内容审核红线

### 3.1 零容忍内容（直接封号）

| 类型 | 说明 |
|------|------|
| 政治敏感 | 涉及政治人物、政治体制、政策批评等 |
| 色情低俗 | 裸露、性暗示、低俗内容 |
| 赌博诈骗 | 赌博信息、诈骗套路、传销 |
| 谣言虚假 | 编造虚假信息、传播未经证实的消息 |

### 3.2 严重违规内容（限流→封号）

| 类型 | 说明 |
|------|------|
| 诱导分享/关注 | "不转不是中国人""转发保平安"等 |
| 虚假宣传/夸大效果 | 编造数据、夸大产品效果 |
| 抄袭洗稿 | 未授权转载他人内容、洗稿 |
| 违规经营 | 金融、医疗等需资质领域的内容 |
| 站外引流 | 诱导用户跳转到微信外的平台 |

### 3.3 一般违规内容（限流→下架）

| 类型 | 说明 |
|------|------|
| 标题党 | 标题与正文内容严重不符 |
| 诱导点击 | 用虚假悬念骗取点击 |
| 内容质量低 | 空洞无物、纯搬运、拼凑内容 |
| 过度营销 | 硬广口吻、频繁推销 |

## 四、违禁词清单

### 4.1 广告法违禁词

最、第一、唯一、首个、首选、顶级、极品、绝对、全网最、史上最、100%、永不、零风险、国家级、世界级

### 4.2 虚假宣传词

包治、根治、药到病除、见效快、零差评、销量冠军、万人疯抢、抢疯了、立竿见影

### 4.3 诱导分享词

不转不是中国人、转发保平安、看到必转、为家人转起、不转后悔

### 4.4 金融/医疗专项违禁词

- 金融：保本保息、稳赚不赔、零风险投资、年化XX%
- 医疗：治疗、治愈、处方、临床验证、医学认证、根治

## 五、标题规范

### 5.1 允许的标题技巧

- ✅ 设置悬念：「为什么越来越多的人开始用XX？」
- ✅ 数据驱动：「5个方法帮我提升了3倍效率」
- ✅ 干货承诺：「从零开始搭建XX，这篇就够了」
- ✅ 真实体验：「用了半年XX，说说我的感受」

### 5.2 禁止的标题写法

- ❌ 纯标题党：「震惊！」「太可怕了！」「不敢相信！」
- ❌ 虚假悬念：标题承诺的内容正文没有兑现
- ❌ 夸大数字：「99%的人不知道」「1亿人都在用」

## 六、排版与格式规范

### 6.1 微信HTML规范

微信 HTML 渲染规则详见 `rules/brand-guidelines.md` 第六章。发布前须确认 HTML 遵守该规范。

### 6.2 图片规范

- 封面图尺寸：900×500 px（详见 `rules/brand-guidelines.md` 第四章）
- 正文图片宽度不超过600px
- 图片不得含有水印（除原创作者水印）
- 不得盗用他人图片

### 6.3 原创与转载

- 原创文章可标注「原创」标签
- 转载必须获得授权并标注来源
- 引用他人内容需明确标注出处

---

## 七、其他注意事项

### 7.1 发布频率

- 建议每天不超过1篇推送
- 多图文推送不超过8篇
- 避免短时间内频繁发布

### 7.2 互动规范

- 评论区不得放任违规内容
- 不得用机器人刷评论
- 不得诱导用户分享到朋友圈

### 7.3 特殊领域

- 金融类内容需要相关资质
- 医疗类内容需要相关资质
- 教育类内容需注意不违规宣传
- 涉及未成年人内容需特别谨慎
